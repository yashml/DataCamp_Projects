# A Network Analysis of Game of Thrones

In this project I will analyze the network of characters in Game of Thrones and how it changes over the course of the books.

### Dataset: 

- Dataset consists of 5 csv files which can be viewed [here](https://github.com/yashml/DataCamp_Projects/tree/master/A%20Network%20Analysis%20of%20Game%20of%20Thrones/datasets). 

### Language and Packages:

- Python: pandas, networkx

### Skills:

- Data Manipulation
- Data Visualization
- Importing and Cleaning
- Network Analysis

### New Learnings:

- Using *DataFrame.from_records()* to convert record ndarray to DataFrame.

- Using *DataFrame.idxmax()* to return index of first occurrence of maximum over requested axis.
