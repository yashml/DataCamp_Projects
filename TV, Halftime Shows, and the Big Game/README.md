# TV, Halftime Shows, and the Big Game

In this project I load, clean, and explore Super Bowl data in the age of soaring ad costs and flashy halftime shows.

### Dataset: 

- The dataset consists of 3 csv files : tv.csv , super_bowls.csv, halftime_musicians.csv which can be viewed [here](https://github.com/yashml/DataCamp_Projects/tree/master/TV%2C%20Halftime%20Shows%2C%20and%20the%20Big%20Game/datasets).

### Language and Packages:

- Python: pandas, Matplotlib, Seaborn

### Skills:

- Data Manipulation <br>
- Data Visualization  <br>
- Importing and Cleaning 

### New Learnings:
- Improving spacing between subplot using *plt.tight_layout()*
